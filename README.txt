EL proyecto fue realizado en kotlin.
Se utilizó mi dispositivo Android Samsung S10 por lo que las interacciones son dependientes de la disposición de mi celular.
Todos los eventos se realizan por wireless commands
Se asumen configuradas las variables del sistema necesarias para ejecutar adb en la consola, así como conocimiento de cómo obtener la dirección ip del celular a utilizar
Se supone el celular no tiene aplicaciones usadas recientemente
Se supone se puede ejecutar proyectos kotlin en el computador a probar. 
Se supone que los dispositivos están en la misma red.
